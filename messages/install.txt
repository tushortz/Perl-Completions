Perl Completions
================

A Sublime text package that displays several completions for the PERL language methods

Usage
-----

normal typing of method names shows methods for modules. Note that the amount of * displayed next ot the func specifies how many arguments the function takes.

Contributing
------------

All contributions are welcome. fork Perl Completions on Github at <https://github.com/tushortz/Perl-Completions> and create a pull request. Any suggestions or bugs, please let me know.
